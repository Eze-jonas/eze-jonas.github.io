Data Science / Machine Learning / Data Engineering

  • Credit Card Eligibility Prediction
    This project developed predictive models to determine credit card eligibility using traditional machine learning algorithms (Decision Tree, Logistic Regression) and a deep learning approach (Artificial Neural Network). Feature selection techniques, including Genetic Algorithm (GA) and Principal Component Analysis (PCA), were applied to improve model performance. Among the tested methods, the Decision Tree model using GA-selected features showed the best results. The selected model was deployed via a RESTful API built with Flask, featuring an interactive web form enhanced with JavaScript and CSS, and hosted on the cloud using Render. This work demonstrates the practical application of machine learning to financial services and provides a foundation for more advanced, scalable credit eligibility systems.
